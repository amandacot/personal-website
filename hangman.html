Hangman!

Everyone knows the game Hangman, so I decided to make my own online version of the game! I utilized user input and let players select their own word, or I let them pick a category. I learned how to code a keyboard and how to make each letter work when playing hangman! Dashed lines appear for the word you're trying to guess. For each correct letter guess made, I learned how to make that letter appear on the corresponding dashed line. For each mistake made, I learned how to draw a different shape on the canvas to make my hangman.
