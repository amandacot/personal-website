Ice Cream

I scream, you scream, we all scream for ICE CREAM!

This simple ice cream cone was one of the first Java projects I ever made. However, it wasn't as simple as it looks. By making this cone, I learned how to make buttons function, and how to make each button do something different. The "blue" button taught me how to change the background color. The "strawberry" button taught me how to change colors of objects by changing the first scoop from white to pink. The sprinkles button was one of the most complicated ones. I had to learn how to set locations of each sprinkle. Since they're lines,I learned how to set each X and Y coordinate of the beginning and end points. I also learned how to add a label (it adds "ice cream shop" underneath the buttons), how to add a cherry on top (creating a circle), and how to change the color of another object by making the chocolate scoop turn orange for mango. So, this ice cream cone may look simple, but it taught me a lot about different features of Java.
